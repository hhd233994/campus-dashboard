## 模板二：详细版（面试用）

### 校园消费数据管理平台 | 独立开发

**项目时间**：2026.03 - 2026.05  
**技术栈**：Spring Boot 3、MyBatis-Plus、Redis、MySQL、JWT、EasyExcel、ECharts

**项目背景**：
为解决校园内消费记录分散、统计分析困难的问题，开发了一站式消费数据管理平台。

**核心功能**：
1. **用户认证**：JWT无状态认证 + BCrypt密码加密
2. **权限控制**：基于AOP的RBAC权限管理（ADMIN/USER）
3. **消费管理**：CRUD + 多维度筛选 + 分页查询 + Excel导出
4. **数据统计**：ECharts可视化 + Redis缓存优化
5. **安全审计**：AOP自动记录操作日志 + 接口限流

**技术实现**：
- **认证授权**：自定义JwtInterceptor验证Token，@RequireRole注解实现权限控制
- **性能优化**：Redis缓存统计数据（TTL 1小时），数据库索引优化（user_id、create_time、category）
- **安全防护**：BCrypt密码加密，@RateLimit注解实现滑动窗口限流
- **数据持久化**：MyBatis-Plus LambdaQueryWrapper实现类型安全查询

**项目成果**：
- 缓存命中率 > 80%，平均响应时间 < 100ms
- 单实例QPS可达450+，支持并发用户数100+
- Docker一键部署，开发生产环境一致
