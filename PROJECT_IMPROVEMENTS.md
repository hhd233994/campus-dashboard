## 🔍 项目现状评估

### 优势
1. **完整的功能实现**：用户认证、消费记录管理、数据统计、Excel导出、操作日志等
2. **企业级安全机制**：JWT认证、BCrypt加密、RBAC权限控制、接口限流
3. **良好的架构设计**：前后端分离、分层清晰、AOP切面编程
4. **完善的部署方案**：支持Maven直接运行、Docker一键部署
5. **规范的代码风格**：统一返回结果、全局异常处理、参数校验

### 技术栈
- **后端**：Spring Boot 3.5.13 + MyBatis-Plus 3.5.7 + Redis + MySQL
- **前端**：原生HTML/CSS/JavaScript + ECharts + Tailwind CSS
- **工具**：JWT、EasyExcel、Knife4j、Lombok

### 核心功能
- 🔐 JWT无状态认证
- 🛡️ RBAC权限控制
- 📊 数据可视化（ECharts）
- 🔍 智能筛选（日期、类别）
- 📥 Excel导出
- 💾 Redis缓存
- 📝 操作日志审计
- 🚦 接口限流

## ⚠️ 注意事项

1. **首次启动前**：确保MySQL和Redis已启动
2. **数据库初始化**：创建空数据库`campus_dashboard`，系统会自动执行schema.sql
3. **JWT密钥**：生产环境务必修改application.properties中的jwt.secret
4. **端口占用**：默认使用8080端口，如有冲突请修改配置
