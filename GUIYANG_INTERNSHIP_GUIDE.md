## 📅 实习时间线

### 春招实习（现在 - 4月）

**适合人群**：大二升大三学生

**时间安排**：
- 2月：准备简历和项目
- 3月：开始投递，参加宣讲会
- 4月：面试 + offer

**建议**：
- 这是最好的时机，竞争相对小
- 即使找不到，也为秋招积累经验

### 暑期实习（5月 - 7月）

**适合人群**：大三学生

**时间安排**：
- 5月：集中投递
- 6月：面试高峰期
- 7月：入职

**建议**：
- 重点关注朗玛、满帮等大公司的暑期实习项目
- 提前1-2个月准备

### 日常实习（全年）

**适合人群**：有时间即可

**特点**：
- 随时可以入职
- 要求相对较低
- 适合积累第一份实习经验

## 💼 投递渠道

### 1. 内推（最推荐）

**如何获取内推**：
- 问学长学姐（贵师大计算机学院有很多在贵阳工作的）
- GitHub上找贵阳的开发者
- 技术社区（掘金、CSDN）认识的人
- LinkedIn联系目标公司员工

**优势**：
- 简历直达HR，不被筛选掉
- 面试通过率提高50%+
- 可以了解内部情况

### 2. 公司官网

**操作步骤**：
1. 访问公司官网
2. 找到"加入我们"或"校园招聘"
3. 注册账号，填写简历
4. 选择实习岗位投递

**优势**：
- 信息准确可靠
- 可以追踪投递状态

### 3. 招聘平台

**推荐平台**：
- Boss直聘（回复率高）
- 拉勾网（互联网专用）
- 实习僧（实习专用）
- 智联招聘（传统企业多）

**技巧**：
- 完善在线简历
- 主动打招呼，不要只投简历
- 每天保持活跃

### 4. 校园渠道

**关注渠道**：
- 贵师大就业信息网
- 计算机学院通知群
- 校园招聘会
- 老师推荐

## 🎤 面试准备

### 技术面试（1-2轮）

**常问问题**（参考INTERVIEW_QA_SIMPLE.md）：

1. **Java基础**
   - HashMap和HashTable的区别？
   - ArrayList和LinkedList的区别？
   - String、StringBuilder、StringBuffer的区别？

2. **Spring框架**
   - Spring IOC和AOP是什么？
   - @Autowired和@Resource的区别？
   - Spring Bean的生命周期？

3. **数据库**
   - MySQL索引的数据结构？
   - 什么情况下索引会失效？
   - 事务的隔离级别有哪些？

4. **Redis**
   - Redis为什么这么快？
   - 缓存穿透、击穿、雪崩怎么解决？
   - Redis持久化机制？

5. **项目相关**
   - 请介绍一下你的项目
   - JWT是如何实现的？
   - 项目中遇到的最大挑战是什么？

**准备方法**：
- 阅读INTERVIEW_QA_SIMPLE.md
- 理解原理，不要死记硬背
- 用自己的话表达

### HR面试（1轮）

**常问问题**：

1. 请做一个自我介绍
2. 为什么想来我们公司？
3. 你能实习多久？
4. 你的期望薪资是多少？
5. 你有什么问题想问我吗？

**回答技巧**：

**自我介绍**（1分钟）：
```
面试官好，我叫XXX，是贵州师范大学计算机专业大三学生。

我熟练掌握Java和Spring Boot，独立完成了一个全栈项目：
校园消费数据管理平台，使用了Spring Boot、Redis、MySQL等技术。

我对后端开发很感兴趣，希望能贵公司实习，学习企业级开发流程。

谢谢！
```

**为什么来我们公司**：
```
我了解到贵公司在XXX领域很有实力，技术栈和我学习的方向很匹配。
我希望能在实战中提升自己的技术水平，为公司创造价值。
```

**期望薪资**：
```
贵阳的实习薪资一般在150-300元/天，我接受公司的标准。
我更看重学习机会和成长空间。
```

**反问环节**：
- 实习期间会有导师带吗？
- 主要做什么类型的工作？
- 团队有多少人，技术栈是什么？

## 💡 实用建议

### 1. 关于学历

**贵师大的优势**：
- 贵州省内认可度高
- 计算机学院在本地有校友资源
- 很多贵阳公司有贵师大员工

**如何应对**：
- 强调项目经验和实践能力
- 展示学习能力和主动性
- 不要自卑，能力比学校重要

### 2. 关于经验

**没有实习经验怎么办**：
- 突出个人项目（就是你现在的这个项目）
- 强调自学能力
- 展示对技术的热情

**第一份实习最重要**：
- 不要挑剔，先入行
- 积累3-6个月经验后，跳槽更容易

### 3. 关于技术

**贵阳公司主要用**：
- Java + Spring Boot（最多）
- MySQL + Redis
- Vue.js（前端）
- Linux + Docker

**你的项目完全匹配**，不用担心！

### 4. 关于心态

**常见焦虑**：
- "我的学校不够好" → 贵阳本地公司很认可贵师大
- "我没有实习经验" → 每个人都是从0开始的
- "我怕面试过不了" → 多面几次就有经验了

**正确心态**：
- 每次面试都是学习机会
- 被拒绝很正常，不要灰心
- 坚持投递，总会有offer
